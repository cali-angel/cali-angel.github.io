Welcome to my CIT 82 Project 0

Class Description:

Survey of contemporary web technologies in authoring, distributing and browsing. Topics include: web servers, scripting, protocols, E-Commerce, information architecture, website traffic analysis, search engine optimization, and website usability.

Overview of what I leared:

This class really opened my eyes as far as the different possibility's and ways of doing things in the web development industry. I liked it since i'm pretty inexperienced with web design and the different technologies involved with it and that make development better. And as the name of this course implies, there was an emphasis for technology. I mean, the web basically can't exist without the use of it. The main thing about this course that impacted me is that it wasn't really focused on just one set of tools or gadgets, but rather it covered and introduced us to a whole variety of things that is involved with the web. At first, I was a bit overwhelmed, but I soon realized that it's good to see all the things I could learn and add in my skill set. And of course I can't learn it all at once, but I can make it a goal to learn them eventually so that I can become a well rounded developer. This class was very enjoyable and is a good way to get one that is starting to look into web development prepared for his or her future.
