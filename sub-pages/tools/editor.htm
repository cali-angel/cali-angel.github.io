Hands-On - Select your Editor

In this assignment, we were tasked to pick out a code editor that we liked. The default one we would use in class was VSCode but if there was another that we chose, it was fine.

Overview

I chose VSCode since I was introduced to it last semester by Rio in CIT 15. I enjoyed how it was pretty cool to use and since I already had it installed on my computer, I decided to stick with it. Plus, because it is made by windows, its probably going to stick with the latest trends in tech and its also going to have a good support system as all windows products do. So I started to download the back up and sync from google, and that was pretty simple as well. I just had to choose what files that i wanted to back up on my drive. However, because use the email I used to sign up only for school, i chose not to back up any files from my computer. I did have the CIT 82 folder on my drive synced with my computer. I think that is a really smart idea because now i can put any file i need to upload to my drive from my file explorer. Before, i had to put the file on a folder i made on my computer, then go to drive, and then move it over. Now it has gotten much simpler. Also, It installed shortcuts on my desktop to google docs and more so i can get to my files and docs faster. I really should have done this sooner.
